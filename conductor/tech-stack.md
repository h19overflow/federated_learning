# Technology Stack

## Backend
- **Language:** Python 3.12+
- **Framework:** FastAPI
- **Machine Learning:** PyTorch, PyTorch Lightning
- **Federated Learning:** Flower (flwr)
- **Database:** PostgreSQL with pgvector (via SQLAlchemy/psycopg2)
- **Orchestration:** LangChain, LangGraph (for Agentic Systems)
- **Deployment:** Docker, docker-compose

## Frontend
- **Framework:** React (Vite, TypeScript)
- **Styling:** Tailwind CSS
- **UI Components:** Radix UI, Lucide React
- **Visualization:** Recharts, Plotly (as inferred from project files)

## Development & Infrastructure
- **Package Management:** uv
- **Testing:** pytest
- **Logging/Monitoring:** Weights & Biases (wandb), TensorBoard
